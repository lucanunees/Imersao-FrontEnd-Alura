Quando construímos a estrutura básica de tags do HTML, inserimos meta tags dentro da tag <head>. 
As tags meta determinam os metadados, que são as informações sobre dados de um documento HTML. 
Logo abaixo, podemos visualizar os metadados que colocamos no código do projeto Portfolio.

 <meta charset="UTF-8">
 <meta http-equiv="X-UA-Compatible" content="IE=edge">
 <meta name="viewport" content="width=device-width, initial-scale=1.0">

O metadado http-equiv="X-UA-Compatible" content="IE=edge" é utilizado para configurar a página web no Edge,
 para que ela esteja sempre em sua versão mais recente.

O metadado charset="UTF-8" é empregado para repassar aos navegadores qual é o formato de codificação de 
caracteres utilizado naquele documento.
