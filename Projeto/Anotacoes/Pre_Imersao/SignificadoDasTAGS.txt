
 == Siginificado das TAGS.

<!DOCTYPE html>
<html lang="pt-BR"> -- Defini a lingua que iremos utilizar.
<head>
    <meta charset="UTF-8"> -- Seria a códificação que iremos utilizar (caracteres).
    <meta http-equip="X-UA-Compatible" content="IE=edge"> -- Ele utiliza a ultima versão do navegador especificado.
    <meta name="viewport" content="width=device-width, initial-scale=1.0"> -- Defini a escala inicial da página (basiado no dispositivo que será utilizado, celular ou computador etc).
    <title>Portfolio</title>
</head>
<body>
    
</body>
</html>

 <header></header> -- Seria o cabelhaço da pagina
 <main></main> -- Seria o conteudo principal.
 <body></body> -- Seria o corpo da página.
 <html></html> -- Seria a página HTML. 
 <!DOCTYPE html> -- Seria a versão do HTML, no caso HTML 5.
 <footer></footer> -- Seria o rodapé da pagina.
 <h1></h1> -- É utilizado para criar Títulos.
 <strong></strong> -- É utilizado para destacar um conteúdo.
 <p></p> -- Seria um parágrafo.
 <a></a> -- Seria uma Tag angora, que irá me redirecionar para algum site ou algo do tipo.
 <img> -- Essa Tag não fecha, é utilizada para inserir uma imagem na tela.
 <link>	-- Essa Tag não fecha, é utlizada para linkar um arquivo css com a página.
 <link rel="stylesheet" href="Style.css"> -- rel significa tipo de relacionamento e href o caminho aonde esta o arquivo.
